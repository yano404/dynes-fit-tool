Dynes fit tool
==============

![screenshot](assets/screenshot.png)

## Requirements
- numpy
- scipy
- pandas
- plotly
- dash
- dash-bootstrap-components

## Usage
Run the app with
```
$ python app.py
```
and visit http://localhost:8050/ in your web browser.
